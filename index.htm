Contrast

Contrast is a powerful design concept. It simply means difference. When creating a webpage, one should no when to use the same image perhaps, or whether to differ the two images.

Repitition

Repitition is used to make the viewer easily comprehend your website. Such s having light text with a dark background or dark text with a light background. If a web page has a light background with light text that is BAD Repitition.

Alignment

A Web-Page designer will need to understand the meaning of alignment before he/she can even begin developing the URL page. Alignment is a key thing that we must understand. We use Alignment to organize the text within the page to make it easier on the eyes of the viewer.

Proximity

Proximity is another key element of building a web page. A web page designer needs to realize whether he/she has to move perhaps an image further or closer. Using this key word will allow people to easily decipher what kind of images and text is on your page.
